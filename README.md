# 项目简介
本项目基于CNN实现表面肌电信号（sEMG）手势识别任务。针对传统sEMG连续幅值图像存在噪声冗余、手势特征区分度低的问题，本文设计了一种幅值分层量化表征方法，将连续肌电幅值离散化为多级灰度特征，有效抑制无效噪声、放大不同手势的特征差异。同时搭建原始图像与量化分层图像双通道融合框架，搭配通道注意力模块，通过多组完备消融实验验证各模块的有效性与兼容性。

# 核心亮点
1. 创新幅值分层量化策略：对标准化sEMG信号进行幅值离散分层，过滤微小肌肉波动噪声，大幅提升手势特征区分度，相比原始图像基线准确率稳定提升11%~12%。
2. 双通道特征融合机制：融合原始连续幅值细节特征与分层量化结构特征，实现信息互补，进一步提升模型识别精度与泛化能力。
3. 完备消融实验体系：控制变量验证量化分层、特征融合、通道注意力的独立增益，实验结果可复现、可信度高。
4. 解耦式代码设计：数据预处理与分层消融实验分离，生成缓存文件后可快速迭代测试不同参数，运行效率高。

1. 幅值分层量化是性能提升核心模块：相较于原始连续肌电图像，分层量化表征可稳定提升11%~12%识别准确率，有效优化特征可分性。
2. 双通道融合具备显著互补增益：结合原始幅值细节与分层结构特征，可在单分层输入基础上进一步提升模型性能。
3. 通道注意力适配双通道场景：单一输入（纯原图/纯分层）无法发挥注意力优势，仅双通道融合输入可激活注意力模块的正向增强效果。

# 运行教程 / Run Instructions
1. 基础消融实验：直接运行 main.py，自动完成数据集预处理、希尔伯特图像转换、6组对照消融实验，生成数据缓存文件。
2. 分层数量消融实验：待缓存文件生成后，运行 **ablation_layers.py**，自动测试不同量化层数的模型性能。

# 项目功能 / Project Features
1. 完整sEMG预处理流程：滑动窗口截取、带通/陷波滤波、通道独立标准化
2. 肌电信号时空转换：基于希尔伯特曲线的一维时序信号转二维图像
3. 幅值分层量化：连续肌电信号离散化编码，强化手势特征差异
4. 可配置CNN模型：支持开关通道注意力、自定义Dropout、超参数
5. 全方位模型评估：准确率、宏F1值、单类别精确率/召回率/F1值
